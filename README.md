## 🚀 Sobre o projeto

O **Ficha Automática Skyfall** é uma aplicação web desenvolvida em React que permite aos jogadores criar fichas de personagens de forma rápida, intuitiva e automatizada.

A proposta do sistema é eliminar processos manuais e repetitivos na criação de fichas, centralizando tudo em uma interface simples e funcional.

---

## ✨ Funcionalidades

* 📌 Criação automatizada de fichas
* 🧙 Definição de atributos do personagem (classe, legado, trilha, etc.)
* 📊 Organização clara das informações
* ⚡ Interface rápida e responsiva
* 🌐 Acesso direto via navegador (sem necessidade de instalação)

## 📦 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/MVNSouza/ficha-automatica-skyfall-react

# Acesse a pasta
cd ficha-automatica-skyfall-react

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
